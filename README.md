\# Insurance Pricing Forecast Project



\## Overview

This project predicts insurance prices using Machine Learning.



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- Matplotlib



\## Features

\- Insurance price prediction

\- Regression modeling

\- MAE and RMSE evaluation

\- Prediction visualization

\- Saved ML model



\## Files

\- insurance\_data.csv

\- insurance\_model.pkl

\- predict\_insurance.py

\- notebook.ipynb



\## How to Run



Install libraries:



pip install pandas scikit-learn matplotlib notebook joblib



Run notebook:



jupyter notebook



Run prediction script:



python predict\_insurance.py
